\# NDVI Analysis - Warangal, Telangana



Vegetation health analysis using Sentinel-2 satellite imagery and Python.



\## Formula

NDVI = (NIR - Red) / (NIR + Red)



\## Data Source

Sentinel-2 L2A, Copernicus Data Space, June 16 2026, Warangal, Telangana



\## Libraries

rasterio, numpy, matplotlib



\## Run

pip install rasterio numpy matplotlib

python ndvi.py
